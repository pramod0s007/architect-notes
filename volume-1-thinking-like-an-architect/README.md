# Volume 1 — Thinking Like an Architect

Before studying individual patterns, engineers need to understand **how architects think**.

This volume establishes the mental model that all subsequent volumes build on: start from pressure, not patterns.

## Read Before Any Other Volume

These three papers are the foundation. Every pattern in Volumes 2–5 is an application of the framework introduced here.

The question this volume answers: **How do architects decide when and which pattern to apply?**

Answer: they identify the pressure first.
